Resume is a clean, super flexible and fully responsive Bootstrap HTML Template (try resizing your browser), suited for interior business websites, and users who want to showcase their work on a neat portfolio site. The Template is built on top of the fabulous Bootstrap framework and can be used with any CMS if needed because the CMS template conversion would be easier due to the well documented and neat coding convention. It comes with a documentation both in written and video form so that anyone with very basic knowledge of HTML and CSS, can modify the code easily to suit their need.
